# Manual Pessoal do Core Digital Aragão

## Camadas do sistema
- Público: home, soluções, planos, sobre, contato.
- Cliente: login, área `/app`, acesso aos módulos contratados.
- Admin: `/admin`, dashboard e gerenciador de módulos.
- Módulos: sistemas plugáveis em `/modulos/...`.
